# Tabela Compras

## Project Overview

The Tabela Compras project is designed to help users manage their shopping lists efficiently. The application provides features for creating, updating, and deleting items from a shopping list, along with additional functionalities to enhance the overall user experience.

## Features
- Create and manage shopping lists
- Add items with quantities and notes
- Remove items from the list
- User authentication
- Data persistence through local storage or a back-end API

## Setup Instructions

To run this project locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/nakedsend-bit/tabela-compras.git
   cd tabela-compras
   ```

2. **Install dependencies**:
   Ensure you have [Node.js](https://nodejs.org/) installed. Then run:
   ```bash
   npm install
   ```

3. **Start the development server**:
   ```bash
   npm start
   ```
   Open your browser and navigate to `http://localhost:3000` to view the application.

4. **Build for production**:
   When you're ready to deploy, you can build the project:
   ```bash
   npm run build
   ```

5. **Testing**:
   To run tests, use the following command:
   ```bash
   npm test
   ```
